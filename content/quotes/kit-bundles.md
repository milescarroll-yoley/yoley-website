Kit Bundles save time on jobs you quote for repeatedly. Instead of entering each line item manually, bundle them into a named package and add the whole thing in one tap.

Set up bundles in **Settings → Kit Bundles**. Each bundle can mix labour, materials and travel lines. When you add a bundle to a quote you can still edit individual lines — the bundle is a starting point, not a lock.
